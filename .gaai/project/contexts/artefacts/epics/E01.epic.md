# Epic E01: Market Regime Module

## Purpose

Fund managers can see the current market regime (Red / Yellow / Green) at a glance, automatically computed from live indicator data, so they have a reliable and consistent daily market-context read without manual calculation or cross-referencing.

---

## Scope

- Initialize and maintain a persistent historical indicator database (minimum 1 year, append-only, never deleted)
- Fetch live indicator data: SPX price, SPX 12-day EMA, SPX 25-day EMA, MMTH
- Detect bearish and bullish divergences between SPX and MMTH
- Classify market regime using the 8 validated conditions
- Display regime with supporting indicator values and the active condition driving it
- Manual and/or automatic data refresh

---

## Out of Scope

- Historical regime tracking or charting
- Cross-module signal alignment/interaction (future Epic)
- User authentication or access control
- Mobile-specific layout
- Alerts or notifications

## Success Metrics

- Regime is computed and displayed correctly for all 9 conditions (8 named + bullish divergence absorbed when SPX is already above 12d EMA)
- All indicator values are sourced automatically — no manual input required
- Fund managers can read the regime and its driving condition in under 5 seconds
- Error states are explicit — no silent failures or incorrect regime labels shown
