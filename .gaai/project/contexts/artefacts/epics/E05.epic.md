# Epic E05: Cross-Signal Regime Section

## Purpose

Fund managers can see a synthesised "Regime" read at the top of the Macro tab that aggregates four cross-module signals — Growth, Inflation, Monetary Policy Stance, and Global Credit Creation — and derives two actionable outputs (Market Narrative and Long Bias), so they can arrive at a single cohesive positioning read without manually cross-referencing four separate data sources.

---

## Scope

- Fetch and persist four new data series not covered by existing epics: SOFR (FRED weekly avg), IORB (FRED weekly avg), EURCHF 5-day performance (barchart), GDP YoY% (FRED quarterly)
- Compute all four Regime criteria and two derived outputs from DB data (reusing E02 and E03 pipeline outputs where available)
- Display the Regime section above Growth and Inflation on the Macro tab
- Update the dashboard prototype (prototypes/index.html) to include a hardcoded Regime section

---

## Out of Scope

- Modifying any E02 or E03 story — E05 is a consumer of their DB outputs only
- Historical charting of Regime signals (current-value display only)
- Alerts or push notifications for regime changes
- Automated regime refresh scheduling (on-demand only, consistent with other modules)

---

## Stories

> Stories are tracked in the backlog. This list is descriptive only.

- E05S01: Fetch and persist SOFR, IORB, EURCHF, and GDP YoY%
- E05S02: Compute Regime signals (Growth, Inflation, MPS, GCC, Market Narrative, Long Bias)
- E05S03: Display Regime section on Macro tab
- E05S04: Update dashboard prototype to include Regime section

---

## Success Metrics

- All four Regime criteria compute correctly from live DB data with no manual input
- Market Narrative and Long Bias derive deterministically from the four criteria
- Regime section renders above Growth and Inflation in the Macro tab
- Any unavailable input surfaces as "Unavailable" — no silent wrong outputs
- Prototype reflects the same layout as the live Regime section

---

## Dependencies

- E02S02 — Net Liquidity series must be persisted before E05S02 can derive WoW change
- E03S02 — WoW% changes for T5YIE, US02Y, US10Y, DXY, USDJPY, USDCNH must be computed before E05S02
- E03S04 — Macro tab display must exist before E05S03 can add a section to it
- E05S03 — live display spec must be complete before E05S04 updates the prototype
